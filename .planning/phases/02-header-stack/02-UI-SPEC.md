# Phase 2 — UI Design Contract: Header & Stack

> Visual and interaction contract for a GitHub Profile README.
> No web framework, no CSS, no React. All "UI" is shields.io badge URLs, readme-typing-svg parameters,
> and GitHub-Markdown-compatible HTML structure.
> Generated by gsd-ui-researcher, verified by gsd-ui-checker.

---

## Design System

| Property | Value |
|----------|-------|
| Tool | none (GitHub Markdown renderer) |
| Preset | not applicable |
| Component library | none |
| Icon library | simple-icons (via shields.io `logo=` param) |
| Font | not applicable — renderer-controlled |

### GitHub Rendering Constraints

| Rule | Detail |
|------|--------|
| Centering | `<div align="center">` only — `style=` is stripped by GitHub sanitizer |
| Block wrapper | Each centered section wrapped in `<div align="center">...</div>` |
| HR dividers | Raw `---` on its own line between every major section |
| No `style=` | Confirmed zero occurrences in Phase 1 output — must stay zero |
| SVG backgrounds | Must use `background=00000000` (transparent) — not `background=transparent` |
| Badge style split | `for-the-badge` on standalone rows (this phase); `flat-square` inside `<td>` (Phase 3 only) |

## Spacing Scale (README Structure)

There is no pixel-based layout. "Spacing" in a GitHub README is structural: blank lines and section separators.

| Token | Markdown Element | Usage |
|-------|-----------------|-------|
| section-break | `---` (HR divider) | Between every major section |
| block-gap | One blank line | Between `<div>` and its content |
| badge-gap | One space ` ` | Between badges on the same line |
| group-gap | One blank line | Between emoji label and badge row |

4. `---`

## Typography (Badge Labels & Text Conventions)

There is no CSS typography. "Typography" here means badge label text conventions and shields.io label encoding.

| Role | Convention | Example |
|------|-----------|---------|
| Badge label | Title Case | `Python`, `FastAPI`, `Tailwind CSS` |
| Scikit-learn label | Exact: `Scikit--learn` | Double hyphen = one literal hyphen in shields.io path |
| Slash in label | URL-encode: `%2F` | `shadcn%2Fui` |
| Space in label | Use `_` underscore | `Tailwind_CSS` |
| Emoji labels | UTF-8 direct in Markdown | `📊 **ML / Data Science**` |
| Label bold | `**text**` in Markdown | Rendered as bold in GitHub |

### Group Label Format (D-08, D-09)

Each badge group has a Markdown label on its own line directly above the badge row, inside the same `<div align="center">`:

```
📊 **ML / Data Science**

[![badge1](url)](link) [![badge2](url)](link) ...
```

## Typing SVG — Complete Specification

### Endpoint

`https://readme-typing-svg.demolab.com` — the image source root. The old `herokuapp.com` endpoint is dead (HEADER-03, STATE.md).

### Complete URL (ready to paste)

```
https://readme-typing-svg.demolab.com?font=Fira+Code&size=22&duration=3000&pause=700&color=58A6FF&background=00000000&center=true&vCenter=true&width=600&lines=Data+Scientist;ML+Engineer;Full-Stack+Developer
```

### Parameter Reference

| Parameter | Value | Source | Note |
|-----------|-------|--------|------|
| `font` | `Fira+Code` | Discretion | Monospace — fits tech/terminal aesthetic |
| `size` | `22` | Discretion | 22px — visible at GitHub profile width without overflow |
| `duration` | `3000` | D-01 (fast/snappy) | 3000ms total cycle per line (type + erase); longest line "Full-Stack Developer" is ~20 chars — results in ~150ms/char of pure typing time at 3000ms total, which renders as fast and snappy |
| `pause` | `700` | D-02 (minimal) | 700ms pause between roles — constant-motion feel |
| `color` | `58A6FF` | D-06, HEADER-02 | Accent color — no `#` prefix in URL param |
| `background` | `00000000` | HEADER-02, VISUAL-02 | Transparent — 8-digit hex RGBA encoding is safer than the keyword `transparent` |
| `center` | `true` | HEADER-03 | Centers text within SVG width |
| `vCenter` | `true` | HEADER-03 | Vertically centers text within SVG height |
| `width` | `600` | Discretion | 600px fits GitHub profile column on desktop |
| `lines` | `Data+Scientist;ML+Engineer;Full-Stack+Developer` | D-04, HEADER-01 | Semicolon-separated; `+` for spaces within a line |
| `cursor` | `true` (implicit default) | D-03 | Blinking cursor — reinforces the typing effect |

### Embedding Pattern

```html
<div align="center">
  <img src="https://readme-typing-svg.demolab.com?font=Fira+Code&size=22&duration=3000&pause=700&color=58A6FF&background=00000000&center=true&vCenter=true&width=600&lines=Data+Scientist;ML+Engineer;Full-Stack+Developer" alt="Typing SVG" />
</div>
```

This block replaces `<!-- PHASE-2-HEADER -->` on line 1 of README.md.
